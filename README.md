## 🇺🇸 English

### 📚 What's Inside?

**dcode** is a complete guide to:
- 🔐 **Cryptography:** Caesar, Vigenere, RSA, AES
- 🏆 **CTF Challenges:** Capture The Flag exercises
- 🛠️ **Security Tools:** John the Ripper, Hashcat, Hydra
- 🔍 **Forensics:** File analysis, memory dumps

### 📂 Structure

```
dcode/
├── 01-cryptography/     # Encryption, Decryption, Hashes
├── 02-ctf-challenges/   # Practice challenges
├── 03-tools/            # Security tool guides
└── 04-forensics/        # Digital forensics
```

## 🇮🇷 فارسی

### 📚 چه چیزهایی اینجاست؟

**dcode** یه راهنمای کامل برای:
- 🔐 **رمزنگاری:** سزار، ویژنر، RSA، AES
- 🏆 **چالش‌های CTF:** تمرینات پرچم‌گیری
- 🛠️ **ابزارهای امنیتی:** John the Ripper، Hashcat، Hydra
- 🔍 **پزشکی قانونی:** تحلیل فایل، دامپ حافظه

### 📂 ساختار

