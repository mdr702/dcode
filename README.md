## 🇺🇸 English

### 📚 What's Inside?

**dcode** is your complete hands-on guide to the world of cybersecurity, cryptography, and CTF (Capture The Flag) competitions.

| Section | Description |
|---------|-------------|
| 🔐 **Cryptography** | Caesar, Vigenere, RSA, AES, Hashing (MD5, SHA-1, SHA-256) |
| 🏆 **CTF Challenges** | Real-world Capture The Flag exercises with step-by-step solutions |
| 🛠️ **Security Tools** | Hashcat, John the Ripper, Hydra, Nmap, Wireshark |
| 🔍 **Forensics** | File analysis, memory dumps, steganography, metadata extraction |

### 🚀 Quick Start Guide

```bash
# 1. Clone the repository
git clone https://github.com/mdr702/dcode.git

# 2. Navigate to the folder
cd dcode

# 3. Start with Cryptography
cd 01-cryptography
cat caesar-cipher.md

# 4. Try a CTF challenge
cd ../02-ctf-challenges
cat crypto-challenge.md

# 5. Learn a tool
cd ../03-tools
cat hashcat-guide.md
```
