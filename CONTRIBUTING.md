# 🤝 Contributing to dcode

Thank you for considering contributing to **dcode**! 🎉

## 🎯 Ways to Contribute

1. **Add new tutorials** - Share your cryptography or CTF knowledge
2. **Fix typos** - Improve the documentation
3. **Translate** - Add more languages
4. **Report issues** - Found a bug? Let us know!
5. **Suggest ideas** - Open an issue with your suggestion

## 📝 How to Contribute

1. Fork the repository
2. Create a new branch: `git checkout -b my-feature`
3. Make your changes
4. Commit: `git commit -m "Add my feature"`
5. Push: `git push origin my-feature`
6. Open a Pull Request

## 📋 Guidelines

- Keep it simple and beginner-friendly
- Add both Persian and English versions
- Test your commands before submitting
- Update the README if needed
